There are three zip files in this folder

documentaion.zip - contains the clear details about the game

game.zip - contains the optimised game files  ( not source)

source.zip - contains the source files for the game including assets , js ,json and html files


Nasıl çalıştırırsın

Prod derle ve Android’e senkronla:
cd D:\siteler\wwwroot\games\memory_match\source

npm run build:android

Android Studio’da aç:
npm run cap:open:android

Yalnız web build:
npm run build
